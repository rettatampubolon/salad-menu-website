# Salad Menu Website

## Deskripsi Program
___
Website ini menunjukkan menu-menu salad dan juga resep-resepnya yang memiliki konten sebagai berikut:
* main page
* home page

Selain itu, resep menu bisa diakses apabila kita mengklik foto/box dari menu tersebut.
